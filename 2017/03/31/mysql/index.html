MySQL登录和退出

  • MySQL登录
参数 描述
-D, –database=name 打开指定数据库
–delimiter = name 指定分隔符
-h, –host=name 服务器名称
-p, –password[=name] 密码
-P, –port=# 端口号
–prompt=name 设置提示符
-u, –user=name 打开指定数据库
-V, –version 输出版本信息并且退出
  • MySQL退出
    • mysql > exit;
    • mysql > quit;
    • mysql > \q;
